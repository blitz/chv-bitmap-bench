# Cloud Hypervisor dirty_log() Optimization

This repo is a benchmarking harness around the code in [Cloud
Hypervisor](https://www.cloudhypervisor.org/) that scans dirty bitmaps during
live migration.

## Let's go

Just use `cargo bench`. Have fun!
